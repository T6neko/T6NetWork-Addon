# T6NetWork-Addon
T6NetWorkServerで使用されているアドオンです。主にロビーサーバー向けのアドオンです。

## 機能一覧
- /t6:lobby  
このコマンドで設定した場所にTPできます。config.jsで変更できます。
- /t6:list  
このコマンドで管理者が設定したサーバー一覧を表示できます。config.jsから変更できます。
- /t6:server  
このコマンドでリスト以外のサーバーに行けるフォームを出します。
- /t6:serverwith  
このコマンドでリスト以外のサーバーへ他プレイヤーと一緒に行くことができるフォームを出します。
- 参加時指定場所TP  
サーバー参加時に管理者が設定した場所にTPします。config.jsで変更できます。
- 参加時に特定のメッセージ送信  
参加時にそのプレイヤーにメッセージを送信できます。config.jsで変更できます。
