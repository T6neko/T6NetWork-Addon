# T6NetWork-Addon

Minecraft統合版の **Script API** を利用した、サーバー運営支援アドオンです。  
主にロビーサーバーでの利用を想定しており、プレイヤーのサーバー移動や初期地点への誘導を強力にサポートします。

---

## 🚀 主な機能

### 🛠 コマンド
- **/t6:lobby** 管理者が設定したロビー地点に一瞬でテレポートします。
- **/t6:list** サーバーリストを表示します。ここから別のサーバーへ簡単に移動できます。
- **/t6:server** リストにないサーバーへ移動するための専用フォームを表示します。
- **/t6:serverwith** 他のプレイヤーを誘って一緒に別サーバーへ移動できる、連携フォームを表示します。

### ⚙️ 自動処理
- **初期スポーンTP** プレイヤーが参加した際、あらかじめ設定した座標へ自動的にテレポートさせます。
- **ウェルカムメッセージ** 参加したプレイヤーに対して、個別にカスタマイズしたメッセージを自動送信します。

---

## 🛠 導入・設定方法

### 動作条件
- Minecraft 統合版 (Bedrock Edition)
- **ベータAPI (Beta APIs)** の実験的なゲームプレイを有効にしてください。

### 設定の変更
アドオン内の `config.js` を書き換えることで、テレポート先やメッセージの内容を自由に変更できます。
