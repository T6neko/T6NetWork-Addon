# T6NetWork-Addon

Minecraft統合版の **BDS (Bedrock Dedicated Server)** 専用の運営支援アドオンです。  
Script APIを利用し、特にロビーサーバーにおけるプレイヤーの誘導やサーバー間移動の自動化に特化しています。

> 本アドオンは、**公式サーバーソフトウェア (BDS)** での動作を前提としています。
> シングルプレイや通常のマルチプレイではなく、サーバー運営での利用を推奨します。

---

## 🚀 主な機能

### 🛠 サーバー運営コマンド
- **/t6:lobby** 設定済みのロビー地点へ即座にテレポート。
- **/t6:list** カスタマイズ可能なサーバーリストを表示し、ワンタップで別サーバーへ転送。
- **/t6:server** リスト外のサーバー情報を自由に入力して移動できるフォームを表示。
- **/t6:serverwith** 他のプレイヤーと一緒に、指定したサーバーへグループ移動。

### ⚙️ 自動システム
- **初期スポーン固定** サーバー参加時に、管理者が指定した座標へ全プレイヤーを自動テレポート。
- **カスタムウェルカムメッセージ** 参加プレイヤーごとに個別のメッセージを自動表示。

---

## 🛠 導入・設定方法

### 動作条件
- **Minecraft 統合版 (Bedrock Edition)**
- **公式サーバーソフトウェア (BDS)**
- **ベータAPI (Beta APIs)** の実験的なゲームプレイを有効化してください。

### 設定の変更 (`config.js`)
アドオン内の `scripts/config.js` を編集することで、以下の設定が可能です。
- テレポート座標
- サーバーリスト
- ウェルカムメッセージの内容
